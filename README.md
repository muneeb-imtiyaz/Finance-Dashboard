# 💰 Finance Dashboard

 A modern and responsive finance dashboard built using React and Tailwind CSS.


## Key Highlights

* Clean and modern UI
* Interactive charts (Recharts)
* Role-based UI (Admin / Viewer)
* Transaction filtering and insights
* Fully responsive design

##  Architecture

* Modular folder structure (`ui`, `store`, `helpers`)
* Context API for state management
* Reusable components

##  Tech Stack

* React (Vite)
* Tailwind CSS
* Recharts
